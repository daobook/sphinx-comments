Hypothesis¶

Hypothesis 是一个中心化的网络服务，允许你在网络上对任意的网页进行评论和注释。

Hypothesis 在这个页面上被激活。你可以通过点击本页右上角的 < 按钮看到网络覆盖。

Activate hypothes.is¶

你可以通过在你的 conf.py 文件中添加以下内容来激活 hypothes.is ：

comments_config = {
   "hypothesis": True
}

这将为你的文档添加一个 hypothes.is overlay。这个扩展只是在你的 Sphinx 网站上激活了 hypothes.is 的 javascript 包。这将导致显示 hypothes.is 覆盖，让你的读者在有问题时可以登录并评论你的文档。

当你建立你的文档时，你会看到屏幕右侧的 hypothes.is 叠加。
