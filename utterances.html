Utterances¶

Utterances 是一个建立在 GitHub issue 之上的评论引擎。它在你的页面上嵌入了一个评论框，用户（拥有 GitHub 账户）可以用它来提出问题。这些问题会成为你选择的仓库中 GitHub issue 的评论”

Utterances 在这个页面上被激活。你可以在页面内容的底部看到评论框。点击 “登录” 按钮，你就可以发表评论了！

激活 utteranc.es¶

你可以通过在你的 conf.py 文件中添加以下内容来激活 utteranc.es ：

comments_config = {
   "utterances": {
      "repo": "github-org/github-repo",
      "optional": "config",
   }
}

注解

你可以为 utterances 传递可选的额外配置。参见 关于你的选项的 utterances 文档。

接下来，按照 utteranc.es 的配置说明。

当你建立你的文档时，页面现在会在底部有一个评论框。如果读者通过 GitHub 登录，他们将能够发表评论，然后映射到你的 GitHub 存储库中的 issue 上。
