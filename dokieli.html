Dokieli¶

警告

Dokieli 是实验性的，现在的行为可能与预期的不一样！

dokie.li 是一个客户端编辑器，用于分散的文章发布、注释和社交互动。Dokieli 在这个页面上被激活。你可以通过点击本页右上角的汉堡包菜单看到网络覆盖。

激活 dokie.li¶

你可以通过在你的 conf.py 文件中添加以下内容来激活 dokie.li：

comments_config = {
   "dokieli": True
}

接下来，遵照 dokie.li 的配置说明。

当你建立你的文档时，页面现在将有一个活跃的 dokieli 叠加。
